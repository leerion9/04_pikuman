# images 폴더

앱에서 사용하는 이미지 파일을 둡니다.

- 스플래시용 로고 (실제 스플래시 이미지는 assets/splash/ 사용)
- 게임 타일 그림
- 버튼·아이콘 등
- **app_icon.png**: 런처 아이콘 원본(1024×1024 권장). 적용: 프로젝트 루트에서 `dart run flutter_launcher_icons` (설정은 pubspec.yaml `flutter_launcher_icons`). 플레이 콘솔용 512 아이콘은 별도 제출

해상도별 폴더(2.0x, 3.0x)를 쓰려면 pubspec.yaml assets에 경로를 추가하세요.
