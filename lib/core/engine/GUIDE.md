# core/engine/ 폴더 안내

노노그램(네모로직) 퍼즐의 핵심 계산·검증 로직 모음입니다.
(Phase 2에서 구현 예정)

## 파일 목록 (구현 예정)

| 파일 | 역할 |
|------|------|
| `nonogram_model.dart` | 퍼즐 데이터 모델 (JSON ↔ Dart 변환) |
| `clue_calculator.dart` | 이진 배열 → 행·열 클루(숫자 힌트) 계산 |
| `puzzle_validator.dart` | 플레이어 입력과 정답 비교·검증 |

## 핵심 개념

- **클루(Clue)**: 각 행·열에서 연속으로 채워진 칸의 개수 목록
  - 예: `[0,1,1,0,1,1,1,0]` → 클루는 `[2, 3]`
- **검증**: 플레이어가 채운 칸 배열과 solution 배열을 비교
