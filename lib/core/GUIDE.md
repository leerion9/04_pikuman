# core/ 폴더 안내

여러 화면(feature)에서 공통으로 사용하는 핵심 기능 모음입니다.
특정 화면에만 종속되지 않는 코드는 모두 여기에 둡니다.

## 하위 폴더

| 폴더 | 역할 |
|------|------|
| `database/` | SQLite DB 헬퍼 및 퍼즐·진행 상태 DAO |
| `engine/` | 노노그램 클루 계산·정답 검증 로직 |
| `network/` | 서버에서 퍼즐 JSON을 다운로드하는 API 통신 코드 |
| `services/` | 오디오, 광고, 설정 등 전역 서비스 |
| `widgets/` | 배너 광고 등 공통 위젯 |
